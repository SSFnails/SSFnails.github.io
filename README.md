# SSF nails

Приложение записи для мастера маникюра: дашборд мастера + публичная страница самозаписи.

- Фронт: React (Vite) + Tailwind, мобайл фёрст
- База: Supabase (Postgres + Auth + Storage), данные только в БД
- Хостинг: Vercel

## Запуск с нуля (минут 15)

### 1. Supabase

1. Зайди на [supabase.com](https://supabase.com), создай бесплатный проект (регион ближе к дому, пароль БД сохрани).
2. Открой **SQL Editor**, вставь целиком содержимое файла `supabase/schema.sql` и нажми **Run**. Это создаст таблицы, прайс, права доступа и хранилище для референсов.
3. Создай аккаунт мастера: **Authentication → Users → Add user → Create new user**. Укажи её почту и пароль, поставь галочку **Auto Confirm User**.
4. Возьми ключи: **Project Settings → API**. Нужны `Project URL` и `anon public` ключ.

### 2. Локальный запуск

```bash
cp .env.example .env    # вставь туда URL и anon ключ из шага 4
npm install
npm run dev
```

- Дашборд мастера: http://localhost:5173 (вход по почте и паролю из шага 3)
- Страница записи для клиентов: http://localhost:5173/book

### 3. Деплой на Vercel

1. Залей папку в GitHub репозиторий (приватный).
2. На [vercel.com](https://vercel.com): **Add New → Project**, выбери репозиторий. Фреймворк Vite определится сам.
3. В **Environment Variables** добавь `VITE_SUPABASE_URL` и `VITE_SUPABASE_ANON_KEY`.
4. Deploy. Ссылку вида `https://xxx.vercel.app/book` отдай клиентам, сам дашборд по корневой ссылке.

## Как это устроено

- **Слоты (окошки)** мастер выставляет сама на конкретные дни, никакого авто-расчёта по длительности. Статусы: свободно, занято, заблокировано. Запись автоматически занимает слот, отмена освобождает.
- **Заявки клиентов** приходят со статусом "новая" и ждут подтверждения мастера. Слот при этом уже занят, чтобы не было двойных записей.
- **Доход** считается только по записям со статусом "выполнена".
- **Правила мастера в интерфейсе:** флаг аллергии (дома кот) подсвечивается в записи и карточке клиента, предупреждение после 2х коррекций подряд, тумблер "адрес отправлен" и блок клиентов на завтра, подсказка "пора напомнить" через 2-3 недели носки.
- **Экспорт** записей и клиентов в CSV (открывается в Excel) на вкладке "Ещё".
- Цены услуг редактируются на вкладке "Ещё", в БД ничего не захардкожено.

## Бэкапы

Supabase на бесплатном тарифе хранит бэкапы сам. Дополнительно раз в месяц можно нажимать "Записи в Excel" и "Клиенты в Excel" на вкладке "Ещё" и складывать файлы в облако.
